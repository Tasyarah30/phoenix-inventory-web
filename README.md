# phoenix-inventory-web
# 📦 Rancang Bangun Sistem Manajemen Stok Barang Berbasis Web pada Phoenix Grocery

## 📖 Deskripsi
Proyek ini merupakan aplikasi sistem informasi berbasis web yang dirancang untuk membantu proses pengelolaan stok barang pada Phoenix Grocery. Aplikasi ini mampu mencatat data barang masuk dan keluar, serta menyediakan informasi stok secara real-time untuk memudahkan pengambilan keputusan dan efisiensi operasional toko.

---

## 🧩 Fitur Aplikasi
- ✅ Login sistem untuk admin
- 📦 Manajemen stok barang (tambah, edit, hapus)
- 📥 Pencatatan barang masuk
- 📤 Pencatatan barang keluar
- 🧾 Laporan data barang
- ⏰ Pencatatan tanggal kadaluarsa barang
---

## 🛠️ Teknologi yang Digunakan
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: PHP (Native)
- **Database**: MySQL
- **Tools**: XAMPP / Laragon (untuk local development)
